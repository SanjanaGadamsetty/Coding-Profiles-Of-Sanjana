## 🌸 About

**Sanjana's Coding Profiles** is a beautifully crafted personal website that serves as a central hub for all my coding and professional profiles.

Inspired by the **Japandi aesthetic**, this project blends Japanese minimalism with Scandinavian warmth to create a calm, elegant, and interactive experience.

Instead of sharing multiple profile links, this website brings everything together in one place.

---

## ✨ Features

🌸 Elegant Japandi-inspired design

🌞 Light Theme

🌙 Dark Theme

🔲 Tile View

📋 List View

🔍 Instant Search

🔗 One-click Profile Access

📋 Copy Profile Links

📱 Fully Responsive

💾 Saves Theme & View Preferences

✨ Smooth Animations
